传统图像处理方案二：基于 Canny 边缘检测与形态学的泪河高度测量

1. 简介
本代码 (005-canny+closing.py) 实现了基于传统计算机视觉算法的眼表泪河提取与泪河高度（Tear Meniscus Height, TMH）的自动测量。本方案不依赖深度学习模型，无需数据标注与训练，主要利用灰度梯度特征与数学形态学规则实现端到端的物理指标量化，具有极高的可解释性，运行效率达到毫秒级。

2. 核心算法管线
代码严格按照以下四个阶段的串行逻辑执行：
（1）图像读取与增强：加载彩色眼表图像并转换为灰度图，应用 CLAHE（限制对比度自适应直方图均衡化）增强局部细节。
（2）参考基准轴定位：通过阈值分割与轮廓拟合提取瞳孔（或 Placido 盘中心）区域，计算绝对圆心坐标 (cx, cy)，并以此确立 TMH 测量的全局垂直基准轴。
（3）泪河区域提取：
采用 Canny 双阈值边缘检测提取图像的高频梯度边缘。
采用 形态学闭运算 (Morphological Closing)，利用特定尺寸的矩形结构核（如横向延展），强制连通断续的泪河边缘，生成闭合的候选轮廓。
（4）定量高度测量与物理换算：沿提取的垂直基准轴搜索泪河上、下边界像素，求出差值 TMH_px。随后利用前置步骤拟合出的角膜/瞳孔参考直径，基于 11.5mm 的临床角膜直径先验公式，将像素高度转换为绝对物理高度 TMH_mm。

3. 环境依赖
代码基于标准 Python 科学计算栈构建，请确保安装以下依赖包：
opencv-python (cv2)
numpy
matplotlib
安装命令：
pip install opencv-python numpy matplotlib

4. 运行说明
修改路径：运行前，请务必将代码中的 img_path 变量修改为你本地测试图像的绝对路径（代码中默认使用了 cv2.imdecode 与 np.fromfile 的组合，已原生支持包含中文路径的文件读取）。
执行脚本：
python 005-canny+closing.py
终端输出：程序将在终端输出 TMH像素 与换算后的 TMH毫米 数值。
可视化结果：代码末尾集成了结果可视化逻辑（红色原点为测量基准心，红色竖线为测量轴，绿色覆盖区域为提取出的泪河 Mask），可视化图片矩阵可通过修改代码保存至本地或直接 cv2.imshow 查看。
